What is Rango Exchange?

Rango is a routing and aggregation protocol for on-chain and cross-chain swaps, combining decentralized exchange liquidity and bridge paths in a wallet-connected interface. The official docs describe a chain-agnostic routing engine for EVMs, Bitcoin, Cosmos, Solana, Starknet, and other ecosystems.

How Rango Exchange works

  1. Set routeSelect the source token and network, then choose the destination asset and chain.
  2. Connect walletUse the wallet required for the source network and add a destination wallet when the route needs it.
  3. Review quoteCheck output, slippage, route steps, network fees, and any route-specific protocol or Rango fee shown in the app.
  4. Confirm swapApprove only the transaction you understand, sign in your wallet, and track the request inside the app.

Before you swap

Route discovery

The app can compare same-chain and cross-chain paths across DEX aggregators, bridges, and liquidity providers, then surface available routes for the selected pair.

Chain-agnostic coverage

Rango's routing docs reference support across EVM, Bitcoin, Cosmos, Solana, Starknet, and other ecosystems, with availability depending on tokens and providers.

Security posture

Rango is non-custodial, so wallet approval stays with the user; published audit reports are listed in the official docs.

Rango Exchange FAQ

What is Rango Exchange used for?

It is used to route same-chain and cross-chain swaps from one token or network to another through a wallet-connected, non-custodial interface.

Is Rango Exchange the official app?

This landing page is a routing page; the official app is app.rango.exchange, and users should verify that domain before connecting a wallet.

Does Rango Exchange custody my funds?

Rango describes its flow as non-custodial, meaning funds remain controlled from your wallet and transactions require wallet approval before execution.

What fees can appear on Rango Exchange?

Quotes can include network gas, swapper or bridge fees, and a Rango fee depending on route; the fee docs explain the categories.

Why might a Rango Exchange route show no path?

A route may be unavailable when the amount is outside provider limits, the asset is unsupported by available bridges or DEXs, or a real-time quote source is delayed.
